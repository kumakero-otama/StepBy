# analog ページ構成

このフォルダはアナログ時計画面です。

## ファイル一覧

### `Index.html`
- アナログ時計ページ本体。
- 役割:
  - 時計ダイヤルと針要素（時・分・秒）
  - ホームへ戻るリンク（`/home/Index.html`）
  - バージョン表示
- 読み込むスクリプト:
  - `analog.js`
  - `/version.js`
  - `/pwa.js`

### `analog.css`
- アナログ時計専用スタイル。
- 主な対象:
  - 円形ダイヤル
  - 針（時・分・秒）の形状と配置
  - ページ全体のレイアウト

### `analog.js`
- 時計針の角度計算と更新。
- 主要処理:
  - 現在時刻から角度を計算（時針・分針・秒針）
  - 針要素の `transform` を更新
  - 1秒ごとに再描画
