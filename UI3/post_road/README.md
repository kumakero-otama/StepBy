# post_road ページ構成

このフォルダは「道情報の投稿」画面です。

## ファイル一覧

### `Index.html`
- 投稿画面本体（HTML/CSS/JavaScriptを1ファイルに同居）。
- 役割:
  - 投稿地点の地図表示（Leaflet）
  - タグ選択UI（既存タグ検索 + 追加）
  - 説明文入力
  - 画像選択（ライブラリ / カメラ）とプレビュー
  - 画像圧縮・サイズ制限チェック
  - 投稿送信
- 主要API:
  - `/api/post-tags`（タグ一覧取得・タグ追加）
  - `/api/config`（画像サイズ制限値の取得）
  - `/api/road-info`（投稿保存）
- 主要処理:
  - URLクエリ（`lat`,`lng`）があれば初期中心に反映
  - 現在地追従トグルで地図中心制御
  - 送信時にタグID・説明文・画像データURLをJSON化してPOST
